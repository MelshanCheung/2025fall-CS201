# 我的学习资料

这个仓库对应2025年秋季学期yhf老师开设的数据结构与算法B，这门课程我主要是1.5倍速听老师分享的录播，辅以较多的课后刷题。

我认为对于期末很重要的练习是来自于力扣的算法题单，这里附上网址：https://leetcode.cn/circle/discuss/RvFUtj/

最后也分享一下我在实习面试的时候被问到的两个数据结构问题，非常简单：
1. 如果用两个list实现stk？
2. 用什么数据结构来维护一组数据，使得能够最快找到最常用的前十个数据？

至于笔试部分，这个学期由于自主命题，考的是从零构建LLM的习题，我在考前使用notebookllm生成了两套模拟题，感觉基本上覆盖了所有考点。
